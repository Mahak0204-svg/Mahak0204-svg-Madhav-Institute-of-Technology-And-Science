Topic 1: Blinking LED using Arduino

1. Introduction

The Blinking LED project using Arduino is a fundamental and widely practiced beginner experiment in electronics and embedded systems. It involves programming an Arduino to turn an LED on and off at regular intervals, demonstrating the use of digital output and timing functions.

This project introduces key concepts such as circuit connections, the use of resistors to protect components, and basic programming structures like setup() and loop(). By mastering this simple task, learners gain confidence to explore more complex Arduino applications.

1.1 LED (Light Emitting Diode)
An LED is a semiconductor light source. It emits light when current flows through it. The longer leg (anode +) connects to a digital pin, and the shorter leg (cathode -) connects to GND. Polarity matters.

3. Circuit Diagram

3.1 Procedure of Connection

  • Connect the long leg (anode) of the LED to digital pin 13 on Arduino.
  • Connect the short leg (cathode) to one end of the 220Ω resistor.
  • Connect the other end of the resistor to GND (ground) on the Arduino.

3.2 Arduino Code

// Blink LED on pin 13

void setup() {
  pinMode(13, OUTPUT); // Set pin 13 as an output
}

void loop() {
  digitalWrite(13, HIGH); // Turn LED on
  delay(1000);            // Wait 1 second
  digitalWrite(13, LOW);  // Turn LED off
  delay(1000);            // Wait 1 second
}

4. Result & Discussion

The LED connected to digital pin 13 blinks on and off every second. This demonstrates digital output control using digitalWrite() and timing using delay(). It repeats indefinitely as coded in the loop() function.
