10 Days Program Schedule

Topic No. Topic Name Brief Summary Applications View
1 Blinking LED Turns an LED on and off at fixed intervals using Arduino. Basic electronics learning, debugging indicators.
2 Traffic Light System Simulates real-world traffic light operation using red, yellow, and green LEDs. Traffic light simulation, automation training.
3 LED Fading (PWM) Uses PWM to gradually increase and decrease LED brightness. Mood lighting, breathing LED effects.
4 Button-Controlled LED LED toggles on/off based on button press input. User input systems, basic control interfaces.
5 LED Chaser Lights up a series of LEDs one by one to create a chasing effect. Decorative lighting, turn signal simulations, visual indicators.
6 Temperature-Activated LED LED turns on when the temperature sensor detects a set threshold. Fire alarms, environmental monitoring.
7 Sound-Activated LED LED responds to sound signals like claps using a sound sensor. Clap switches, interactive displays, simple voice triggers.
8 RGB LED Color Mixing Combines red, green, and blue LEDs to produce different colors using PWM. Mood lamps, digital signage, color testing tools.
9 Motion-Activated LED Lights up when motion is detected using a PIR sensor. Security systems, automatic lighting.
10 Morse Code Transmitter Translates text into Morse code and blinks it through an LED. Educational tools, emergency signaling.
